# 🎬 Netflix Movies & TV Shows Analysis

### 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The analysis aims to uncover trends in content distribution, genres, ratings, release years, and country-wise content availability on Netflix.

### 🎯 Objectives

* Explore the composition of Netflix content.
* Analyze the distribution of Movies and TV Shows.
* Identify popular genres and content ratings.
* Examine content growth over the years.
* Discover country-wise content trends.
* Generate insights through data visualization.

### 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

### 📂 Dataset Features

The dataset contains:

* Show ID
* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Listed In (Genre)
* Description

### 🔍 Exploratory Data Analysis

#### Data Cleaning

* Handled missing values
* Removed duplicates
* Converted date columns
* Processed categorical features

#### Analysis Performed

* Movies vs TV Shows Distribution
* Content Added Over Time
* Release Year Trends
* Top Content-Producing Countries
* Most Common Ratings
* Popular Genres Analysis
* Duration Analysis
* Netflix Content Growth Trends

#### Visualizations

* Bar Charts
* Histograms
* Count Plots
* Pie Charts
* Heatmaps
* Line Charts

### 📊 Key Insights

* Movies make up the majority of Netflix content.
* Content additions increased significantly after 2015.
* The United States contributes the largest share of content.
* Drama and International Movies are among the most popular genres.
* TV-MA is one of the most common content ratings on Netflix.
